4. Clone & Status:

(a) Clone  - Cloning a repository on our local machine

        (i) git clone <link>
        (repository >> (<>code) >> HTTPS >> <copy link>)

(b) Status - Displays the state of the Code
        (ii) git status

5. IDK:
(a) untracked: new files that git doesn't yet track
(b) modified: changed
(c) staged: file is ready to be committed
(d) unmodified: unchanged 

6. Add & commit:
(a) git add <file name>
(b) git commit -m "comment"

7. Push Command
(a) push = upload local repo content to remmote repo
        (i) git push origin main
        origin = name & main = branch name GitHub

8. Init Command:
(a) init - Used to create a new git repo
        (i)     git init
        (ii)    git remove add origin <link>
        (iii)   git remove -v (to verify remote)
        (iv)    git branch (to check branch)
        (v)     git branch -M main (to rename branch)
        (vi)    git push origin main
